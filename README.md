# Task Dependency Tracker (INMUTO SDE Intern Assignment)

A task dependency management system where tasks can depend on other tasks.
Includes circular dependency detection, auto status updates, and dependency graph visualization.

## Tech Stack
- Frontend: React 18+, Tailwind CSS
- Backend: Django 4.x + Django REST Framework
- Database: MySQL 8.0+
